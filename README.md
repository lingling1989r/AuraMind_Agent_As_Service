# AuraMind Tools: 企业级Agent落地开源工具集

本仓库是 AuraMind 项目的开源工具专区，会持续更新共享企业级Agent落地的核心工具与实践案例，目前优先开源了 **amk存储权限隔离插件**，聚焦解决多用户Agent服务的存储隔离痛点。

### 项目背景

AuraMind 整体项目聚焦两大方向：基于 Aura Desk 的 Vibe Marketing 内容运营，以及企业级 Agent 落地。我们会逐步把项目中经过实践验证的工具、案例开源到这里：

- 现在已开源：核心`amk`存储插件，解决行业普遍痛点 **「当你把Agent封装为SaaS服务，面向多用户提供服务时，默认没有为每个用户做独立的权限管控和物理存储空间隔离」**——amk会给每个agent分配单独的workspace，从存储层实现物理隔离+权限管控，让你放心开放Agent服务。

- 未来持续更新：会逐步分享**SaaS产品Agent化的落地案例**、配套适配工具等，帮更多开发者少踩企业级Agent落地的坑。

### 已开源核心能力：amk存储隔离插件

- 🛡️ **原生权限+物理隔离**：每个Agent独享独立workspace，从底层满足多租户、多客户场景的安全隔离要求
- 🔌 **轻量易集成**：作为独立存储插件，可以快速和你的Agent项目集成，不需要从头搭建隔离体系
- 🎯 贴合开放MCP生态：解决MCP Skill平民化后的管线管控问题——当人人都可以扩展Agent能力，我们提前做好权限安全规则，就像开车需要交规，让普惠的AI能力用得安心

本仓库 `/tools` 目录下还会陆续更新更多配套存储适配工具，欢迎持续关注~
