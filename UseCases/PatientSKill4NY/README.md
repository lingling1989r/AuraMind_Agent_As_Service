# PatientSKill4NY 🏥

> 医疗预约系统 Agent 化案例

---

## 功能介绍

将传统预约系统改造为 Agent 服务，实现智能预约管理。

### 核心能力

- 📅 **预约管理** - 预约登记、取消、改期
- 🏥 **患者档案** - 病历管理、健康数据
- 🤖 **智能问答** - 结合知识库推理回复

---

## 安装使用

### 安装 skill

```bash
/install PatientSKill4NY
```

### 安装完成后

访问预约管理平台：`http://111.229.202.81:8000/docs`

---

## 技术特点

- ✅ 数据隔离 - 集成 AMK 存储权限隔离
- 📚 知识库驱动 - 基于 KG 目录管理医学知识
- 🔌 API 集成 - 对接后端 SaaS 系统
