# CRM Agent 🔧

> 客户管理系统 Agent 化案例

---

## 功能介绍

将传统 CRM 系统改造为 Agent 服务，通过自然语言管理客户数据。

### 核心能力

- 👥 **客户管理** - 添加、查询、更新客户
- 📊 **数据看板** - 销售数据统计
- 🔍 **智能查询** - 自然语言搜索

---

## 安装使用

### 安装 skill

```bash
/install order-demo-api
```

### 安装完成后

访问 CRM 平台管理后台

---

## 技术特点

- ✅ 数据隔离 - 集成 AMK 存储权限隔离
- 📦 开箱即用 - 标准化 skill 格式
- 🔌 API 集成 - 对接后端系统
